# class_label_maker

## usage
- .jpg 데이터셋이 있는 폴더에 main.py 파일 복제
- 콘솔로 데이터셋 있는 폴더로 이동
   - python main.py [category name]
- [category name]의 하위 클래스들은 main.py의 [category name]_list에서 리스트 형태로 지정

- 'a': 이전 이미지로 이동
- 'd': 이후 이미지로 이동
- 'q': 종료
- 우측 클래스 선택(복수 선택 가능) 후 다른 이미지로 이동하면 같은 폴더에 해당 이미지와 같은 이름의 .txt파일로 레이블링되어 저장됨
